### ***Descripition***
Game-Collection-Tracker is a full-stack web application that allows users to create an account, add video games, and write reviews about them. Users can explore games added by others, share their opinions, and manage their own game collection.
This project demonstrates core full-stack development concepts using Node.js, Express, MongoDB, and EJS, including authentication, routing, CRUD operations, and session management.

### ***Getting Started***
1. Open the project
2. Sign-up (Name - Email -Picture- Password_Confirm Password)
3. Sign-in
4. Adds a new game
5. Writes reviews
6. User views games and reviews
7. Sign-Out
